# @use-barcode-scanner/hooks

Various hooks to use for accessing video devices, getting media streams, displaying video on canvases, and most importantly scanning canvases for barcodes using the built-in BarcodeDetector or a polyfill.

See [@use-barcode-scanner/components](https://github.com/j5bot/use-barcode-scanner-components) `BarcodeScanner` component for an example of using the hook.

Storybook stories for this library are available at https://jonathancook.site/use-barcode-scanner-hooks/
